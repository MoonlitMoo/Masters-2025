# Midcalibration

```python

# This CASA pipescript is meant for use with CASA 6.6.1 and pipeline 2024.1.0.8
context = h_init()
context.set_state('ProjectSummary', 'observatory', 'Karl G. Jansky Very Large Array')
context.set_state('ProjectSummary', 'telescope', 'EVLA')
try:
    hifv_importdata(vis=['24A-411.sb45152792.eb45228190.60344.213862488425.ms'])
    hifv_flagdata(hm_tbuff='1.5int', fracspw=0.01, intents='*POINTING*,*FOCUS*,*ATMOSPHERE*,*SIDEBAND_RATIO*, *UNKNOWN*, *SYSTEM_CONFIGURATION*, *UNSPECIFIED#UNSPECIFIED*')
    hifv_vlasetjy()
    hifv_priorcals(show_tec_maps=False)
    hifv_aoflagger(flag_target='"1331+305=3C286"', aoflagger_file='clean.lua')
    hifv_testBPdcals()
    hifv_semiFinalBPdcals()
    hifv_aoflagger(flag_target='J1041+0610', aoflagger_file='clean.lua', use_corrected=True)
    hifv_aoflagger(flag_target='J1058+0133', aoflagger_file='clean.lua', use_corrected=True)
    hifv_aoflagger(flag_target='J1150+2417', aoflagger_file='clean.lua', use_corrected=True)
    hifv_solint()
    hifv_fluxboot()   
finally:
    h_save()
```

## Checks

- Delay: Good
- Gain phase: Seems nice and continuous, bit harder to tell since longer time scale.

- Fluxboot:
    - J1041+0610 ~1.4 Jy
    - 3C286 ~5.23
    - J1058+0133 ~3, measured at 5~6 but is “highly variable”
    - J1150+2417 ~0.78, measured at ~1 is ok

Flagging looks absolutely fine, basically nothing except for the handful of spikes was removed. Calibrators look great, science looks mostly perfect, some small weather + microwave spikes.

# Final calibration

```python
# This CASA pipescript is meant for use with CASA 6.6.1 and pipeline 2024.1.0.8
context = h_init()
context.set_state('ProjectSummary', 'observatory', 'Karl G. Jansky Very Large Array')
context.set_state('ProjectSummary', 'telescope', 'EVLA')
try:
    hifv_importdata(vis=['24A-411.sb45152792.eb45228190.60344.213862488425.ms'])
    hifv_flagdata(hm_tbuff='1.5int', fracspw=0.01, intents='*POINTING*,*FOCUS*,*ATMOSPHERE*,*SIDEBAND_RATIO*, *UNKNOWN*, *SYSTEM_CONFIGURATION*, *UNSPECIFIED#UNSPECIFIED*')
    hifv_vlasetjy()
    hifv_priorcals(show_tec_maps=False)
    hifv_aoflagger(flag_target='"1331+305=3C286"', aoflagger_file='clean.lua')
    hifv_testBPdcals()
    hifv_semiFinalBPdcals()
    hifv_aoflagger(flag_target='J1041+0610', aoflagger_file='clean.lua', use_corrected=True)
    hifv_aoflagger(flag_target='J1058+0133', aoflagger_file='clean.lua', use_corrected=True)
    hifv_aoflagger(flag_target='J1150+2417', aoflagger_file='clean.lua', use_corrected=True)
    hifv_solint()
    hifv_fluxboot()
    hifv_finalcals()
    hifv_applycals()
    hifv_aoflagger(flag_target='Z3146', aoflagger_file='clean.lua', use_corrected=True)
    hifv_aoflagger(flag_target='RXCJ1115+0129', aoflagger_file='clean.lua', use_corrected=True)
    hifv_aoflagger(flag_target='A1413', aoflagger_file='clean.lua', use_corrected=True)
    hifv_aoflagger(flag_target='A1795', aoflagger_file='clean.lua', use_corrected=True)
    hifv_statwt()
    hifv_plotsummary()
    hifv_exportdata()
finally:
    h_save()
```
