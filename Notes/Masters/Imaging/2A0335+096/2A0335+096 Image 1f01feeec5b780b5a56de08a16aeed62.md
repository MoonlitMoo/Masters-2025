# 2A0335+096 Image

The C-config data has a beam resolution of `Beam : 2.8137 arcsec, 2.02465 arcsec, 9.15468 deg` so I could use a pixel size of 0.5 arc seconds. For a width of 12’ we could go to 1440 pixels, or 1536 (2^9 * 3).

This clips some of the data we had previously which was ~19.2’ wide. So instead I’ll use a width of 20’ as goal which is 2400 pixels. $2^8 \times 3^2=2304$ which is apparently 19.2” exactly.

The data was collected with channel averaging of 16, time averaging of 60s. 

# Dirty Image

```python
tclean(vis=["cconfig.ms", "dconfig_1.ms", "dconfig_2.ms"], 
	imagename='image-dirty',
	imsize=[2304, 2304], cell=['0.5arcsec','0.5arcsec'], 
	specmode='mfs', niter=0, gain=0.1, threshold='1.0mJy',
	deconvolver='mtmfs', pblimit=-1.e-6,
	interactive=True,
	stokes='I', weighting='briggs',robust=0.5,pbcor=False)
```
