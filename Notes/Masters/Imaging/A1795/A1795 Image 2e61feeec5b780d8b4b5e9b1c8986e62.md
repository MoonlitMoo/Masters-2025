# Calibration 3

Last image had 1-2 negative spots that were non-negligible. Continuing to try improve point source model. Lowering threshold, adding extra scale + smallscalebias. Increase sidelobe threshold a lot since its picking up at 2.5. Last RMS was 5.3 uJy. 

This is probably because the c config sees a twirly thing while the d config does not, even though flux is about the same, small scale bias should help with modelling twisty rather than averaging over itnano clea. 

```jsx
tclean(vis=["cconfig-p2.ms", "dconfig1-p2.ms", "dconfig2-p2.ms"],
	imagename='image-p2',
	imsize=[2304, 2304], cell=['0.5arcsec','0.5arcsec'],
```
