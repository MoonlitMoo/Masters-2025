# MS1455+2232 Minihalo

Red shift for MS1455+2232 is z = 0.258, which is 2.67 kpc per arcsec, which is then 1.33 kpc per cell. Beam resolution is ~1.8” which is 3.92 kpc. 

## Observation Data

- **cconfig** 3/2/2024
    - uvwave range  [0, 136000]
- **dconfig 1** 28/2/25
    - uvwave range [0, 35000]
- **dconfig 2** 27/3/25
    - uvwave range [0, 35000]

So 1 year between C and D configurations, 1 month between D configs.

# Check point fluxes

Identified 

- S1 (Minihalo AGN): `"circle[[14h57m15.11s, 22d20m34s], 4arcsec]"`
- S2 (S, annoying): `"circle[[14h57m10.80s, 22d18m44.71s], 2.5arcsec]"`
- S3 (E, head-tail): `"circle[[14h57m08.47s, 22d20m14.5s], 2arcsec]"`
- S4 (ENE): `"circle[[14h57m08.16s, 22d21m09.2s], 3.5arcsec]"`

```python
datasets = ['cconfig', 'dconfig1', 'dconfig2']
labels = ["Minihalo AGN", "S2", "S3", "S4"]
regions = ["circle[[14h57m15.11s, 22d20m34s], 4arcsec]",
           "circle[[14h57m10.80s, 22d18m44.71s], 2.5arcsec]",
           "circle[[14h57m08.47s, 22d20m14.5s], 2arcsec]",
           "circle[[14h57m08.16s, 22d21m09.2s], 3.5arcsec]"]
check_point_flux(datasets, regions, labels, clean_params={"sidelobethreshold": 2.5})
```
