\chapter[Calibration]{Calibration and Imaging}
\label{Chap:Calibration}

The observations to be calibrated are VLA X-band (8-12 GHz) data taken in C and D configurations of 12 galaxy cluster minihalos. The VLA antennas can be arranged in four standard configurations, A through D. A has the longest baselines and best angular resolution, trading off brightness sensitivity. The baseline length decreases towards D configuration, which has the poorest angular resolution, but highest brightness sensitivity. Due to the faint and extended nature of minihalos, the most suitable arrangement is the highly sensitive D configuration. The better angular resolution C configuration observations are added to allow accurate imaging the compact AGN sources. 

The C configuration observations were performed between January and February of 2024, with project code 24A-411, and the D configuration observations were taken during February and March 2025 under project code 25A-157. The frequency range for X-band is 7.977-12.025 GHz with a total bandwidth of 4.048 GHz. This is split into 32 spectral windows with individual bandwidths of 128 MHz. Each spectral window has a spectral resolution of 2 MHz, resulting in a total of 64 channels. An overview of the targeted clusters is given in Table \ref{tab:cluster_list}, while details of the radio observations of each cluster is given in Table \ref{tab:observation_list}.

\section{Imaging}
\begin{itemize}
    \item When we make a raw image from the data, we still have a lot of artefacts from the point spread function, which comes from the beam.
    \item This means we need a more advanced imaging process to remove this to get a clean image. Implemented in a few different ways, we choose tclean in CASA.
    \item Tclean works by modelling the PSF, creating an empty model, find brightest point in residuals, model that as real, subtract model from residuals, repeat 20K times.
    \item This works mostly well, but we can improve results using self calibration.
\end{itemize}

\subsection{Self calibration}
\begin{itemize}
    \item The phase calibration is the most iffy part for the main calibration, so we can make a test image and check the model seems reasonable.
    \item Then use that model to make small phase corrections to our dataset. Then repeat that a few times and these corrections should converge since it is circular.
    \item This is the final calibration steps that we do, phase corrections 20 degrees $\rightarrow$ 1 degree after a few rounds. 
\end{itemize}

    \item Overview of the \texttt{assess\_spw} function used for this.
\end{itemize}
