# A478
This is self-calibrated and imaged with a flux value. I also did some sub-band analysis in the minihalo folder. 

This is split into the 'subband' and 'subband_agn' folders. The plotting for these is done in each folder but there's also some fitting in the main minihalo folder because I am allergic to organising things well. 

I also did a grid search over some robusts and smallscalebias to verify the results were consistent (like with RXJ1720+2638). These are also tossed in the minihalo folder with scripts at the main level and data in the 'grid_search' folder.
