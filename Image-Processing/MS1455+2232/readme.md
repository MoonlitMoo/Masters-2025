# MS1455+2232
This cluster has a self-calibration but not a subtracted image. This is because there is a bright and irritating quasar to the south of the image. It's got very prominent sidelobes due to its dislike of being modelled. These sidelobes domino into a head-tail and then the minihalo. 

There's some debugging per-config minihalo images alongside the joint minihalo image. Otherwise, everything is more or less standard.
