# A1795
Self-calibration is not complete. This cluster is tricky because the BCG is really bright, and has some prominent and twisty radio lobes. The BCG is ~914 mJy at 1.4 GHz (G14), so >10x greater than most of the others in this sample.

The twisty lobe bits were better modelled with scales including 2, but still tend to overcompensate with adding. This makes the cleaning throw big negatives at the model in the 'elbow' of the lobes. Couldn't get deep into the cleanin because the negatives made me grumpy and playing with the parameters wasn't helping a ton. 

Folders are standard.
